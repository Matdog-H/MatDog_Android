## 주요 기술
- 카메라 : 사용자가 앱 화면을 통해 사진 촬영할 수 있도록 한다.
- 앨범연동 : 사용자가 앨범에 있는 사진을 앱으로 가져올 수 있다.
- 객체인식 : 이미지 또는 비디오 상의 객체를 식별하는 컴퓨터 비전 기술로 딥러닝과 머신 러닝 알고리즘을 통해 산출되는 핵심 기술이다.
